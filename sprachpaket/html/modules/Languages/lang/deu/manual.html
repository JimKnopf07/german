Sprachen
  • Sprachen: Das Sprachen-Modul ist in erster Linie an Übersetzer gerichtet. Einzig der Punkt "Sprach Konfiguration" ist für den "normalen" Betrieb von Interesse: Dort wird die Default-Sprache der Website eingestellt, sollen mehrere Sprachen unterstützt werden, kann man dort auch ML (multilanguage) aktivieren. Mit Flaggen kann man dem User ermöglichen seine Sprache im entsprechenden "thelang"-Block über die Landesflagge (anstelle eines Pulldownmenü) auszuwählen.
