Blöcke

Um einen Link, bspw. zu einem neu installiertem Modul, in den Navigationsmenüs hinzufügen zu können, wird vorab kurz auf das Konzept der Blöcke eingegangen. Die Blöcke kann man mit den DesktopWidgets (Taschenrechner, CPU-Temperatur, etc.) der gängigen Betriebssystem vergleichen. Blöcke erweitern die vorhandenen Module um zusätzliche Funktionen, die ergänzend zum Hauptinhalt angezeigt werden. Beim Hauptmenü und Anmeldeformular an den Seitenrändern handelt es sich beispielsweise auch um Blöcke.

Blöcke können für Funktionen genutzt werden, wie die letzten 5 Newsbeiträge, neueste Downloads, letzte Foreneinträge und Mitglieder online. Jedes neu installierte Modul kann unterschiedlich viele neue Blöcke dem System zur Verfügung stellen.

Über die ">Blockadministration können die Menüblöcke bearbeitet werden. Nach dem Klick auf das Bearbeiten-Symbol an der rechten Seite der Zeile, erscheint ein Formular zum Bearbeiten der Links. Dort können Links hinzugefügt, bearbeitet oder gelöscht werden. Jeder Block hat ein eigenes Bearbeitungsformular, indem die modulspezifischen Einstellungen nach eigenen Bedürfnissen und Wünschen angepasst werden können.

Blockpositionen

Die Blöcke können über die func="view"]-->#blockpositions">Blockpositionen in beliebiger Reihenfolge sortiert bzw. positioniert werden. Jede Distribution beinhaltet grundsätzlich die Standard-Blockpositionen "links", "Mitte" und "rechts". Weitere Positionen, wie z.B. für eine Kopfzeile kann folgendermaßen ergänzt werden:

  1. Anlegen einer Blockposition namens 'topblock'
  2. Themetemplates um diese neue Blockposition erweitern
  3. Anlegen eines neuen Blocks mit dem gewünschten Inhalt
  4. Neuen Block die Blockposition 'topblock' zuweisen

Es können beliebig viele Blockpositionen angelgt und über die Themetemplates nach Belieben verteilt werden.
