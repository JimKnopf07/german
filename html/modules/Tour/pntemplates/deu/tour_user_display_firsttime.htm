Was ist Zikula?

Zikula ist ein unter GPL lizenziertes und frei verfügbares Framework mit Benutzerverwaltung und Templating System. Dabei setzt es durchweg auf offene Standards. Zikula ist modular aufgebaut. Wichtige Funktionen werden zentral angeboten und können von zusätzlichen Modulen genutzt werden. Dadurch kann der Leistungsumfang individuell angepasst und nahezu beliebig erweitert werden.

Einen ersten Eindruck über die weitreichenden Möglichkeiten von Zikula konnte man schon bei der Auswahl der Distributionen erhalten. Weitere Informationen über diese Distribution können unter den Link (oben) eingesehen werden. Es stehen diverse Distributionen zur Verfügung, die auf ein bestimmtes Szenario abgestimmt wurden und somit den Einstieg erleichtern. Natürlich können jederzeit zusätzliche Features durch die Installation weiterer Module hinzugefügt werden.
