Willkommen zu PostNuke!

Vielen Dank für die Installation des PostNuke Application Frameworks. PostNuke ist eine frei verfügbare Open Source Software, die unter Berücksichtigung der GPL-Lizenzbestimmungen genutzt werden kann.

Diese Seiten wurden entworfen, um sich mit dem System vertraut zu machen. Die Links sollen Ihnen beim Einstieg helfen. Wählen Sie einfach ein Thema, das für Sie interessiert sein könnte.

Direkteinstieg

Sie können diese Website administrieren, indem Sie auf den Link Administration klicken (in der Regel befindet sich dieser in der oberen linken Ecke). Wenn Sie sich abmelden wird dieser Link wieder verschwinden. Anschließend können Sie sich über den Login-Block (in der Regel auf der rechten Seite) oder über die Login-Seite erneut anmelden.

Sie sind derzeit nicht angemeldet. Sie können sich mit den Benutzerdaten, die während der Installation festgelegt wurden, über den Login-Block (normalerweise auf der rechten Seite) oder alternativ über die Login-Seite anmelden. Nachdem die Anmeldung durchgeführt wurde, erscheint im Hauptmenü (links oben) der Link zur Administration dieser Webseite.

Schließlich, um dieses Tour-Modul zu entfernen und mit der Erstellung der Website zu beginnen...

...gehen Sie einfach in die Administration, klicken auf die Registerkarte "System" und öffnen das Modul "Einstellungen". Um ein anderes Modul auf der Startseite anzuzeigen, ändern Sie die Auswahl im Abschnitt "Startseite" auf ein beliebiges Modul.
